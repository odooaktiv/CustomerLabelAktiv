Customer Labels

The Customer Labels module will allow user to get the customer labels containing customer/vendor/contact’s: name, mail id, phone number, and address. Here are the features of this module:

  • User can get customer labels for customers, vendors, accounting, and contacts.
  • The selected customer/vendor/contact details will be processed for the customer labels generation in PDF format which is printable.
  • User can print 10 customer labels per page at once.

Prerequisites before Installation

No prerequisites are required before the installation.

Configurations

No configurations needed.
