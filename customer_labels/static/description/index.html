Customer Label

The purpose of this module is Create Customer Labels 10 on per page.

You can get customer labels by First selecting the list view in Customers Menu under Sales.And then by Ticking on the customer you need label for and then under PRINT menu you can find "CUSTOMER LABEL".

KEY FEATURES:

  • A PDF file is downloaded under Customer Label name.
  • Can also download VENDOR'S LABEL.
  • Print From Sales / Purchase / Accounting.
  • Print Multiple Customers/Vendors Label Into Single Click.
  • 10 Labels Per Page
